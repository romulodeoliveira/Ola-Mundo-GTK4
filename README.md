# Desenvolvimento Local

### Requisitos Linux

- Rust 1.82.0

### Instalação

Instale o Rust:

```
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

Após a instalação, execute:

```
cargo run
```

Agora instale o GTK4 e a uma dependencia necessária.

```
sudo apt update
```

```
sudo apt install libgtk-4-dev build-essential
```
